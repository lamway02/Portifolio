# Lamway Mogen Portfolio

Modern, responsive React single-page portfolio for Lamway Mogen (System Developer & Product Builder - KOG-POS).

## Features

- Functional React components
- Smooth-scroll top navigation
- Sections: Header, About Me, Projects, Skills, Contact
- Subtle hover and entrance animations
- Mobile-first responsive layout with clean professional styling

## Tech Stack

- React
- Vite
- Plain CSS

## Project Structure

- `src/App.js`
- `src/components/Header.js`
- `src/components/About.js`
- `src/components/Projects.js`
- `src/components/Skills.js`
- `src/components/Contact.js`
- `src/App.css`
- `src/index.css`

## Run Locally

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

Note: Vite 7 recommends Node.js 20.19+.
